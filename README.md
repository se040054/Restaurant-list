## 介紹

這邊有很多好吃的餐廳，當你肚子餓的時候就來看一下吧!

---

## 功能

* 瀏覽餐廳列表
* 查看餐廳詳細資訊
* 在google map 上查看餐廳
* 搜尋餐廳

---

## 啟動專案 

把專案複製到你的電腦上:

` $ git clone https://github.com/se040054/Restaurant-list `

導入專案資料夾:

` $ cd /Restaurant-list `

安裝npm:

` $ npm install `

啟動專案:

` $ npm run start `

成功時，終端機會顯示訊息，請打開瀏覽器進入網址:

` Listening :  http://localhost:3000 `

結束:

> ctrl + c
